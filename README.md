# devCamp2018

## handsonディレクトリ
本セッションで行うハンズオンのコードがあります。

### hands-on3.js
kintone上で別アプリのレコード情報を1件取得するカスタマイズです。

### hands-on4.js
kintone上で別アプリのレコード情報を全件取得するカスタマイズです。<br/>
(1回に取得できるレコード数の上限は100までとなっています。MAX500までは指定可能です。)

### hands-on5.js
タイムカードアプリのカスタマイズです。
